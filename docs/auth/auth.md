## AU0001

- **Message**: invalid credentials
- **Cause**: username or password incorrect
- **Solution**: ask user to re-enter credentials
- **HTTP Status**: 401
- **Category**: business
- **Severity**: low
- **Retryable**: false
- **Tags**: `auth`, `login`
